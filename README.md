**enumeration_priv_esc**

A collection of Linux and Windows enumeration and privilege escalation scripts for use during authorized penetration testing engagements.

**Linux (Bash)**
- System and kernel enumeration
- SUID/SGID binary identification
- Writable file and directory discovery
- Cron job and scheduled task enumeration
- Network interface, routing, and open port enumeration
- Plaintext password and sensitive file search
- Firewall rule inspection

**Windows (PowerShell)**
- Scheduled task executable enumeration and permission analysis
- Service binary permission checks (Full/Modify for Users group)
- Vulnerable executable identification via icacls
- PowerUp output parsing for modifiable service binaries
- Startup program and auto-run detection
- Directory scanning for co-located .exe and .log files
